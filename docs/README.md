## Diseño en Figma

El diseño visual de este proyecto fue elaborado en [**Figma**](https://www.figma.com/), donde se estructuró todo el prototipo tanto para **desktop** como para **mobile**, manteniendo una identidad visual coherente y moderna.

[Ver diseño completo en Figma](https://www.figma.com/design/y075OtskoYggey2rJU9wkn/Cash-Pilot?node-id=107-1169&t=qLwLfJcnlHe7L0vV-1).

---

### Estructura del diseño

El proyecto está organizado por secciones que facilitan su comprensión y desarrollo:

- **Logotipo:** representación principal de la identidad visual del proyecto.
- **Assets:** contiene la **paleta de colores**, la **tipografía** seleccionada y los **recursos descargables** necesarios para la implementación (íconos, imágenes, componentes, etc.).
- **Versión Desktop:** diseño pensado para pantallas grandes, con navegación detallada y distribución completa de los elementos.
- **Versión Mobile:** adaptación del diseño a pantallas pequeñas, optimizando la experiencia de usuario.

---

### Prototipos incluidos

Cada versión (desktop y mobile) contiene prototipos funcionales que muestran el flujo de interacción del usuario:

- **Signup / Registro**
- **Login / Inicio de sesión**
- **Dashboard principal**
- **Ventana de transacciones**
- **Ventana de perfil**
